# Node Net Chat

Chat client in Node.js

# Install

```bash
git clone https://github.com/miguelmota/node-net-chat

cd node-net-chat

npm install
```

## Running chat server

```bash
node index.js
```

## Connecting to chat server

```bash
telnet <server ip address> 9423
```
